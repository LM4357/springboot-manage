## 四、接口测试

### 4.1 测试工具

使用 **Postman** 完成全部接口自动化校验：

| 校验项 | 检查内容 |
|-------|---------|
| 请求参数合法性 | 必填项、类型、长度 |
| 返回数据准确性 | 字段完整性、数据正确性 |
| 接口状态码 | 200/400/401/403/500 |
| 权限拦截逻辑 | 无权限返回 403 |
| 超时异常 | 接口响应时间 < 3s |

### 4.2 接口覆盖

- 用户登录/注册接口
- 商品 CRUD 接口
- 订单操作接口
